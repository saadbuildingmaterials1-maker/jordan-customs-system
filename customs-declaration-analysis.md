## الحقول المطلوبة في النظام:

### 1. معلومات البيان:
- رقم البيان (Declaration Number)
- تاريخ البيان (Declaration Date)
- المركز الجمركي (Customs Center)
- النظام الجمركي (Customs Regime)
- حالة البيان (Status: draft, submitted, approved, rejected)

### 2. معلومات المستورد:
- اسم المستورد (Importer Name)
- الرقم الضريبي (Tax ID)
- رقم الهاتف (Phone)
- العنوان (Address)

### 3. معلومات الشحنة:
- رقم الحاوية (Container Number)
- شركة الملاحة (Shipping Line)
- بلد المنشأ (Country of Origin)
- بلد المصدر (Country of Export)
- ميناء التفريغ (Port of Discharge)
- عدد الطرود (Number of Packages)
- الوزن الإجمالي (Total Weight)
- نوع الحاوية (Container Type)

### 4. الأصناف (Items):
لكل صنف:
- رمز الصنف (Item Code)
- وصف الصنف (Item Description)
- الكمية (Quantity)
- الوحدة (Unit)
- القيمة (Value)
- نسبة الرسوم (Duty Rate %)
- قيمة الرسوم (Duty Amount)
- نسبة ضريبة المبيعات (Sales Tax Rate %)
- قيمة ضريبة المبيعات (Sales Tax Amount)
- الرمز الجمركي HS Code

### 5. توزيع التكاليف:
- قيمة البضاعة (Goods Value)
- الرسوم الجمركية (Customs Duties)
- ضريبة المبيعات (Sales Tax)
- رسوم إضافية (Additional Fees)
- المجموع الكلي (Total Amount)

### 6. توزيع التكاليف على الأصناف:
يجب حساب:
- نسبة كل صنف من القيمة الإجمالية
- توزيع تكاليف الشحن حسب الوزن/الحجم
- توزيع الرسوم الجمركية حسب نوع الصنف
- توزيع ضريبة المبيعات 16%
- التكلفة النهائية لكل صنف

## معادلات الحساب:

### 1. الرسوم الجمركية لكل صنف:
```
الرسوم الجمركية = قيمة الصنف × نسبة الرسوم الجمركية
```

### 2. ضريبة المبيعات لكل صنف:
```
ضريبة المبيعات = (قيمة الصنف + الرسوم الجمركية) × 16%
```

### 3. توزيع تكاليف الشحن:
```
تكلفة الشحن للصنف = (وزن الصنف / الوزن الإجمالي) × تكلفة الشحن الكلية
```

### 4. التكلفة النهائية للصنف:
```
التكلفة النهائية = قيمة الصنف + الرسوم الجمركية + ضريبة المبيعات + تكلفة الشحن + رسوم إضافية
```
